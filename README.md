

#  Airbnb Listings EDA

This project performs **Exploratory Data Analysis (EDA)** on a short-term rental dataset from Inside Airbnb based on listings from Airbnb.

The objective was to clean the dataset, engineer useful features, and analyze patterns in listings, host behavior, and review activity.

---

##  Dataset

The dataset contains **450+ Airbnb listings** with features related to:

* Listing attributes (property type, room type, bedrooms, beds)
* Host characteristics (superhost status, number of listings, host tenure)
* Review statistics and ratings
* Location and availability information

---

##  Data Preprocessing

* Removed irrelevant and redundant columns (IDs, scrape metadata, text fields).
* Handled missing values in review-related features.
* Engineered an **average rating score** feature.
* Detected and analyzed outliers using boxplots.

---

##  Analysis

Performed exploratory analysis to understand:

* Distribution of listing features
* Relationship between occupancy and review activity
* Host behavior and listing patterns

Visualizations were created using **Pandas, Matplotlib, and Seaborn**.
